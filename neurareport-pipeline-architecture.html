NeuraReport V2 — Template & Mapping Pipeline

End-to-end flow: Template creation, AI-powered mapping, contract approval, and artifact generation

Template Creation

  • • Chat Interface: multi-turn LLM conversation
  • • ZIP Import: 5-stage pipeline (write, inspect, allocate, extract, emit)
  • • Duplicate: clone existing template with new ID
  • • Token classification: scalars, row_tokens, totals

AI-Powered Mapping

  • • LLM Call 3: token-to-column mapping with confidence scores
  • • Rich catalog built from DB schema + sample data
  • • SHA256-based caching (PDF + DB + HTML + prompt version)
  • • Multi-DB support: SQLite, PostgreSQL, DuckDB

Contract & Approval

  • • LLM Call 4: generates contract with assumptions + warnings
  • • Validation: token coverage, unknown columns/tokens
  • • Artifacts: contract.json, SQL generators, HTML preview
  • • Final status: template marked "approved"
